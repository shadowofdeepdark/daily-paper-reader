## 摘要
左额叶和颞叶区域的一个网络支持语言理解与产生，执行与词汇提取和语言组合处理相关的计算。本研究探讨：在不同任务情境下，该语言网络对语言刺激的反应在多大程度上保持稳定？其稳定性与一般性多需求（MD）网络的任务敏感性相比如何？参与者（n=52）在六种任务条件下阅读句子和无意义词表，包括被动阅读、在每个刺激后进行记忆探测的阅读，以及需要深层语义参与的问题回答式阅读。句子>无意义词的对比结果在所有任务中均隔离出同一组对语言有反应的体素；这些体素的位置具有个体特异性，突出了个体功能定位的重要性。因此，我们得出结论，语言定位对任务变化具有稳健性。随后我们分析了这些语言反应性体素（语言网络）及一般性MD网络的激活强度和精细激活模式，以测试任务需求是否会调节语言计算和/或招募不同的脑系统。结果显示，语言网络在所有任务下对句子的反应都很强烈，在语义参与度较高的任务中反应稍强。相比之下，MD网络在有任务存在时对句子和无意义词均有反应，这提示在使用包含任务要求的语言实验范式时需谨慎，因为此类范式会激活两个独立的网络。多变量分析进一步表明，语言网络中更容易解码刺激信息，而MD网络中则更易解码任务信息。这些结果表明，在任务驱动的语言理解过程中，语言网络与MD网络执行互补功能：语言网络主要从语言输入中提取信息，而MD网络则确定与任务相匹配的反应。

## Abstract
A network of left frontal and temporal brain areas supports language comprehension and production, implementing computations related to word retrieval and combinatorial linguistic processing. Here, we ask: to what extent are responses to language in this language network stable across task contexts, and how does this stability compare to task sensitivity in the domain-general multiple demand (MD) network? Participants (n=52) read sentences and nonword lists under six task conditions, including passive reading, reading with a memory probe after each stimulus, and reading and answering questions that require deep semantic engagement. The sentences>nonwords contrast isolated the same set of language-responsive voxels across all tasks; the locations of those voxels were participant-specific, highlighting the value of individual-specific functional localization. We therefore conclude that language localization is robust to task variation. We then examined the magnitudes and fine-grained activation patterns in these language-responsive voxels (the language network) and in the domain-general MD network, to test whether task demands modulate linguistic computations and/or recruit a distinct brain system. The language network responded robustly to sentences across all tasks, with somewhat higher responses to semantically engaging tasks. In contrast, the MD network responded to both sentences and nonwords in the presence of a task, which warrants caution when using language paradigms that include task demands, as such paradigms engage two independent networks. A multivariate analysis further revealed that stimulus information is more easily decodable in the language network, whereas task information is more decodable in the MD network. These results suggest that the language and MD networks perform complementary functions during task-driven language comprehension, with the language network primarily extracting information from linguistic input and the MD network determining the appropriate response to the task.

## 一、核心问题与研究背景  

- **研究动机**：  
  人类大脑拥有一组专门用于语言处理的区域——语言网络（language network），主要涉及左额叶和颞叶。该网络在听、读、写及多语种理解中均被证明具有稳定活动，但其是否受任务要求（如注意力、记忆、语义推理等）的调制仍不清楚。  

- **研究核心问题**：  
  1. 语言网络对语言输入的反应在不同任务情境下是否稳定？  
  2. 相比之下，一般性“多需求网络”（MD network，负责任务执行和认知控制）在此过程中表现出何种任务敏感性？  
  3. 若语言网络的反应稳定，是否意味着可普遍使用某一功能定位（localizer）来识别该网络？  

- **科学意义**：  
  该研究旨在阐明语言处理中的“刺激驱动”与“任务驱动”机制界限，为语言神经网络的功能定位和实验范式选择提供理论依据。

## 二、研究方法与技术路线  

- **总体思路**：  
  使用功能磁共振成像（fMRI）记录参与者在不同语言任务下阅读“句子”与“无意义词列”时的脑区反应，并定量比较语言网络与多需求网络的响应模式与空间分布。  

- **关键技术与分析手段**：  
  - **个体化功能定位（Group-Constrained Subject-Specific, GSS）**：在每位受试者脑图中，通过「句子>无词」的对比定义语言网络的功能区（fROI）。  
  - **多版本任务设计**：六种任务版本(V1–V6)，从最简单的被动阅读到需要语义判断的复杂任务。  
  - **多重分析方法**：  
    - **单变量分析（univariate analysis）**：基于BOLD信号变化评估总体反应强度。  
    - **空间重叠与模式相关性分析**：计算任务间fROI一致性（Dice系数）及体素级响应模式相关性。  
    - **多变量解码分析（multivariate decoding）**：利用线性分类器（kNN、SVM、逻辑回归等），测试刺激类型与任务类型在脑区活动中的可分性。  
  - **统计方法**：使用混合效应模型（R包 `lmerTest`）以参与者为随机变量，结合 FDR、Bonferroni 校正。

## 三、实验设计与数据来源  

- **实验对象**：  
  52名健康成年人（平均年龄23.9岁），大部分为右利手，来自麻省理工学院及周边社区。  

- **刺激材料**：  
  - **句子刺激**：语法完整的12词句子，来源于 *Francis & Kučera (1964)* 语料库。  
  - **无词刺激**：通过 Wuggy 软件随机生成的可发音伪词，与句子视觉长度匹配。  

- **任务版本**：  
  | 版本 | 任务类型 | 说明 |
  |------|-----------|------|
  | V1 | 被动阅读 + 按钮提示 | 近似“无任务”条件 |
  | V2 | 难记忆探测 | 判断某词是否出现在句中 |
  | V3 | 易记忆探测 | 判断最后一个词是否重复 |
  | V4 | 问答阅读（匹配语言长度） | 被动阅读但带问题框架 |
  | V5 | 内容理解问答 | 语义判断任务 |
  | V6 | 情感判断 | 评估句子情感正负性 |

- **对比网络**：  
  除语言网络外，还通过空间工作记忆任务（hard>easy）定位一般性多需求（MD）网络。

- **Benchmark与偏基线设置**：  
  以“句子 vs 无词”对比作为标准刺激差异，以“简单 vs 复杂任务版本”衡量任务负荷影响。

## 六、主要结论与发现  

1. **语言网络稳定性**：  
   - 在所有六种任务中，语言网络的反应对句子均显著高于无词（p < .001），表明语言处理具有任务无关性。  
   - 各版本得到的语言区位置一致性（Dice系数平均0.50–0.59）远高于跨个体一致性（约0.13），显示个体化定位的重要。  

2. **任务调制效应**：  
   - 更高认知负荷任务（记忆探测、语义/情感判断）略增强语言区反应，但不改变核心激活模式。  
   - 多需求（MD）网络随任务难度显著增强，尤其在需主动反应条件下。  

3. **解码分析结果**：  
   - 刺激类型（句子 vs 无词）更易在语言网络中解码（准确率≈0.69），  
     而任务类型更易在MD网络中解码（≈0.42）。  
   → 表明两者分工互补：语言网络编码“内容”，MD网络编码“任务”。

## 七、研究亮点与优势  

- **多任务统一分析框架**：在同一实验体系中系统比较六种任务，消除跨研究差异。  
- **个体化功能定位**：强调“每人一图”的精确建模方法，提升结果可靠性。  
- **多层次验证**：结合位置一致性、信号强度、模式分析与解码，形成多角度支撑。  
- **理论贡献**：明确区分刺激驱动的语言处理与任务驱动的认知控制；对语言局部化方法的客观性提供验证。

## 八、不足与局限  

- **任务范围限制**：仅涉及句子层级理解，未覆盖语篇、推理、语用层面。  
- **样本群体单一**：主要为年轻右利手成年人，缺乏儿童、老年、或非母语受试者。  
- **任务与刺激耦合问题**：高级任务不可避免增加对语言内容的再加工，难以完全区分“任务调制”与“语言重处理”。  
- **算力与技术层面限制**：未利用更高时间分辨率或多模态（如MEG/EEG）方法，可能低估瞬时动态响应。
