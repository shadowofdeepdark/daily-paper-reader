## 摘要
动作是否像句子一样被组织？最新证据显示，工具使用与句法理解之间存在相互迁移效应，反映出二者在基底神经节中共享资源。其提出的机制是，将工具嵌入运动计划会增加动作的层级结构，与句子的组织方式相对应。如果这一假设成立，那么语言计算的重叠应特异性地出现在启动和/或伸手抓握阶段，因为此时工具的嵌入会动态更新与目标的关系，而不是当物体已稳定地被工具握持时。四十名法语母语者完成了涉及宾语从句和主语从句的句子理解任务，以及使用工具或徒手抓取并操控物体的运动任务。通过单变量和多变量分析，我们发现，句法复杂句的理解与工具使用中的伸手抓握阶段在基底神经节中引发了重叠且相似的神经编码。这种特定阶段的趋同表明，基底神经节是跨领域层级计算的枢纽，将语言和工具使用中的句法嵌套统一起来。

## Abstract
Are actions organized like sentences? Recent evidence showed reciprocal transfer between tool use and syntactic comprehension, reflecting shared basal ganglia resources for action and language. The proposed mechanism is that embedding a tool into the motor plan increases the hierarchical structure of actions, paralleling the organization of sentences. If so, overlap with linguistic computations should emerge specifically during the initiation and/or reach-to-grasp phase, when tool embedding dynamically updates the relation with the target, rather than when the object is stably held in the tool grip. Forty French native speakers completed a sentence comprehension task involving object- and subject-relative clauses, and a motor task consisting of grasping and manipulating an object either with a tool or with the bare hand. Using both univariate and multivariate analyses, we found that the comprehension of syntactically complex sentences and the reach-to-grasp phase of tool use elicited overlapping and similar neural codes within the basal ganglia. This phase-specific convergence identifies the basal ganglia as a hub for domain-general hierarchical computations, unifying syntactic embedding for language and tool use.

## 二、研究方法与理论框架

- **核心思想**：  
  作者提出一个“跨领域句法假设”（domain-general syntax hypothesis），认为语言句法嵌套与工具使用的层级嵌套在神经层面共享编码通路。  
  关键预测是：句法理解与工具嵌入的动作阶段（如伸手抓握阶段）应在脑区中呈现重叠激活。

- **研究方法**：  
  - 行为任务结合神经影像分析。  
  - 采用**单变量（univariate）**和**多变量（multivariate pattern analysis, MVPA）**两种方法来评估脑区激活与模式相似度。  
  - 使用相似度度量（如余弦相似度 *cosine similarity*）来比较语言任务与动作任务的神经模式。
  - 关注基底神经节（尤其是尾状核 *caudate nucleus*）作为关键区域，假设它参与跨领域计算。

- **关键技术流程**（文字描述）：
  1. 对受试者在两类任务中的神经信号进行采集。  
  2. 在全脑图上通过单变量统计确定句法与动作任务的显著激活区。  
  3. 用 MVPA 计算不同任务条件下模式间的相似度。  
  4. 在感兴趣区域（ROIs）中测试交叉条件的β值与模式重叠情况。  
  5. 对显著性进行多重比较校正。

## 三、实验设计

- **受试者**：40 名法语母语者，无语言或运动障碍。

- **任务设置**：  
  - **语言任务**：区分理解“主语从句”和“宾语从句”。复杂句（宾语从句）代表更高层级的嵌套复杂度。  
  - **动作任务**：抓取与操控物体，有“工具抓握”（tool-use）与“徒手抓握”（free-hand）两种条件。任务分为**启动阶段、伸手抓握阶段、操控阶段**。

- **实验场景与数据类型**：功能磁共振成像（fMRI）数据，结合行为指标（反应时间、准确率）。

- **Benchmark / 对照设计**：  
  各阶段动作条件与语言复杂度条件相互对照，以区分特定阶段（伸手抓握）与特定语法复杂度（宾语从句）间的神经模式是否趋同。

- **对比方法**：无外部模型对比，主要是任务交叉对比与条件差异分析。

## 六、主要结论与发现

- **神经重叠**：  
  发现句法复杂句的理解（宾语从句）与工具使用的“伸手抓握阶段”在基底神经节出现显著重叠激活，尤其是两侧尾状核区域。

- **模式相似度结果**：  
  伸手抓握中的神经模式与句法复杂句的模式高度相似；徒手抓握或简单句不显示这种相似性。

- **阶段特异性**：  
  重叠仅在动态嵌入更新阶段（伸手抓握），而非静态操控阶段发生，说明该阶段的计算需求与句法嵌套相似。

- **总体结论**：  
  基底神经节可能是语言与动作层级结构计算的跨领域核心枢纽（domain-general hub），在人类认知中承担一般层级嵌套计算。
