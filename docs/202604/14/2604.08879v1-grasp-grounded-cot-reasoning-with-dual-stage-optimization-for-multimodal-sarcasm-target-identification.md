## 摘要
多模态讽刺检测（Multimodal Sarcasm Detection）传统上采用二分类范式，而多模态讽刺目标识别（Multimodal Sarcasm Target Identification, MSTI）是一项更具挑战性的任务，要求精确定位诸如文本片段和视觉区域等细粒度目标。现有方法主要依赖于隐式的跨模态对齐，导致可解释性有限且细粒度定位表现不佳。为突破这些局限，我们提出了 GRASP（Grounded Chain-of-Thought ReAsoning with Dual-Stage Optimization for Multimodal Sarcasm Prediction and Target Identification），一个结合视觉语义定位（visual grounding）与显式思维链推理（Chain-of-Thought, CoT）的框架，旨在超越黑盒式的 MSTI 模型。具体而言，我们构建了 MSTI-MAX 数据集，用以缓解类别不平衡问题并增强多模态讽刺线索。我们提出的 Grounded CoT 推理机制显式地在推理轨迹中锚定与讽刺相关的视觉区域，并在预测最终分类标签及讽刺目标前引导模型生成推理依据。此外，我们采用了双阶段结果监督的联合优化策略：首先进行带坐标感知加权损失的监督微调（Supervised Fine-Tuning），随后执行细粒度目标策略优化（Fine-Grained Target Policy Optimization）。大量实验表明，GRASP 在跨模态的细粒度讽刺目标识别任务中显著优于现有基线模型；我们还通过 LLM-as-a-Judge 评估方法定量测量模型内部推理链的质量。我们的数据集与源代码将于 GitHub 上公开。

## Abstract
Moving beyond the traditional binary classification paradigm of Multimodal Sarcasm Detection, Multimodal Sarcasm Target Identification (MSTI) presents a more formidable challenge, requiring precise localization of fine-grained targets such as textual phrases and visual regions. Existing approaches predominantly rely on implicit cross-modal alignment, offering limited interpretability and suboptimal fine-grained localization. To address these limitations, we propose GRASP, Grounded Chain-of-Thought ReAsoning with Dual-Stage Optimization for Multimodal Sarcasm Prediction and Target Identification, a framework that integrates visual grounding with explicit Chain-of-Thought (CoT) reasoning to move beyond black-box MSTI. Specifically, we curate MSTI-MAX, a refined dataset that mitigates class imbalance and enriches multimodal sarcasm cues. We introduce Grounded CoT reasoning, which explicitly anchors sarcasm-related visual regions within the reasoning trajectory and prompts the model to articulate rationales before predicting the final classification labels and sarcasm targets. Furthermore, we employ a dual-stage outcome-supervised joint optimization strategy: Supervised Fine-Tuning with a coordinate-aware weighted loss, followed by Fine-Grained Target Policy Optimization. Extensive experiments demonstrate that GRASP outperforms existing baselines in fine-grained sarcasm target identification across modalities, and an LLM-as-a-Judge evaluation quantitatively measures the quality of internal reasoning chains. Our dataset and source code will be released on GitHub.

## 2. 方法论：GRASP 框架

### 2.1 核心思想
- 提出 **GRASP（Grounded Chain-of-Thought Reasoning with Dual-Stage Optimization）** 框架，核心理念是：
  - 在推理阶段明确地执行“具身（grounded）思维链推理”（Grounded CoT），让模型在“思想链”中显式锚定讽刺相关的视觉区域；
  - 通过双阶段优化机制，结合监督学习和强化学习，逐步增强模型的可解释性与推理精度。

### 2.2 方法结构
- **输入输出形式**  
  - 输入：图像 \(V_i\)、文本 \(T_i\)；  
  - 输出：  
    - 讽刺标签 \( \hat{l} \in \{0,1\} \)，  
    - 文本触发词 \( \hat{k} \)，  
    - 视觉目标框 \( \hat{b} = (x_{\min}, y_{\min}, x_{\max}, y_{\max}) \)，  
    - 推理链 \( r \)，其中推理部分包裹在 `<Think>...</Think>` 标签内，最终答案在 `<Answer>...</Answer>` 标签内。

### 2.3 推理机制：Grounded CoT
- 推理分为两阶段：
  1. **推理阶段（Reasoning Stage）**：  
     模型先生成具身思维链，按结构分析：
     - *视觉描述（Visual Description）*：提取关键视觉对象及其标准化坐标；
     - *文本分析（Text Analysis）*：解析语言含义与情感；
     - *一致性检查（Consistency Check）*：判断视觉与文本的语义一致或冲突；
  2. **输出阶段（Decision Stage）**：  
     根据推理结果输出正式分类、文本目标及视觉框坐标。

### 2.4 双阶段优化策略
- **阶段一：监督微调（SFT）+ 坐标加权损失**
  - 目标：建立任务结构与初步对齐能力。
  - 损失函数：
    \[
    L_{SFT} = -\sum_t \omega_t \log P_\theta(y_t|X,Y_{<t})
    \]
    其中坐标token权重 \(\omega_t = \lambda_{coord} > 1\)，强化视觉定位。
- **阶段二：细粒度目标策略优化（FTPO）**
  - 采用无评论员的强化学习方案（基于GRPO思想），通过采样与奖励学习优化推理链质量。
  - 奖励维度：
    - 格式符合性（R_fmt）；
    - 分类准确性（R_acc）；
    - 视觉定位IoU（R_box）；
    - 文本目标匹配（R_txt）；
    - 冗余惩罚（P_over）。
  - 总奖励函数：
    \[
    R_{total} = \beta_1 R_{fmt} + \beta_2 R_{acc} + \beta_3 R_{box} + \beta_4 R_{txt} - \beta_5 P_{over}
    \]
