## 摘要
持久同调（Persistent Homology）作为拓扑数据分析中的一种方法，能够从数据中提取稳健的多尺度特征。它通过对时间序列的数值施加不同阈值（称为“滤波”（filtration）过程），生成稳定的时间序列表示。我们为时间序列开发了新的滤波方法，并通过将注视序列解释为时间序列，将拓扑学方法引入眼动追踪数据分析中，构建结合拓扑特征与传统统计特征的“混合模型”。我们在阅读眼动数据的阅读障碍检测任务中对该方法进行了实证评估，使用了哥本哈根语料库（Copenhagen Corpus），该语料库包含来自阅读障碍者与非阅读障碍者（母语与第二语言读者）的注视路径。我们的混合模型优于仅依赖传统特征的现有方法，表明持久同调能够捕捉注视序列中所蕴含的互补信息。这些拓扑特征的有效性还体现在其性能可与既有基线方法相媲美。值得注意的是，我们提出的滤波方法优于现有方法。

## Abstract
Persistent homology, a method from topological data analysis, extracts robust, multi-scale features from data. It produces stable representations of time series by applying varying thresholds to their values (a process known as a \textit{filtration}). We develop novel filtrations for time series and introduce topological methods for the analysis of eye-tracking data, by interpreting fixation sequences as time series, and constructing ``hybrid models'' that combine topological features with traditional statistical features. We empirically evaluate our method by applying it to the task of dyslexia detection from eye-tracking-while-reading data using the Copenhagen Corpus, which contains scanpaths from dyslexic and non-dyslexic L1 and L2 readers. Our hybrid models outperform existing approaches that rely solely on traditional features, showing that persistent homology captures complementary information encoded in fixation sequences. The strength of these topological features is further underscored by their achieving performance comparable to established baseline methods. Importantly, our proposed filtrations outperform existing ones.

## 一、核心问题与研究背景

- **研究动机**：阅读障碍（dyslexia）是一种常见的学习障碍，会显著影响阅读效率和理解能力。早期准确检测对教学干预至关重要。  
- **技术背景**：传统的阅读障碍检测通常基于眼动追踪（eye-tracking）数据的统计特征，如注视次数、回视次数、持续时间等。但这些方法可能忽略了注视序列的整体几何与时间结构。  
- **研究目标**：引入 *拓扑数据分析（Topological Data Analysis, TDA）* 中的 **持久同调（Persistent Homology）** 方法，将眼动注视序列看作时间序列，通过拓扑学视角捕捉序列形态与动态模式，从而改进阅读障碍检测模型的表达能力与性能。

## 二、方法论概述

### 核心思想

- 将每个读者的注视轨迹（fixation sequence）转化为时间序列形式：
  - \(T_{j,k} = ((t_1, x_1, y_1), ..., (t_{M_{j,k}}, x_{M_{j,k}}, y_{M_{j,k}}))\)，其中 \(t_i\) 为注视时间，\(x_i, y_i\) 为坐标。
  - 分别取水平与垂直坐标形成两个时间序列：\(T_x\) 与 \(T_y\)。

### 拓扑特征提取流程

1. **时间序列转换**：对 \(T_x\)、\(T_y\) 进行最小-最大归一化。  
2. **持久同调计算**：
   - 对时间序列的图像在二维平面上进行“滤波”（filtration），即逐步改变阈值，观察连通分量（拓扑特征）的出生与消亡。
   - 常规滤波方式为“水平滤波”（horizontal filtration），但此方法不考虑时间轴信息。
3. **提出三种创新滤波方式**：
   - **斜坡滤波（sloped filtration）**：以斜线而非水平线进行扫描，引入时间信息。  
   - **Sigmoid滤波（sigmoid filtration）**与**Arctan滤波（arctan filtration）**：用平滑曲线代替斜线，使滤波过程捕捉时序前后依赖。  
   - 这些滤波生成不同的“持久性图（Persistence Diagram）”，进一步通过 **持久性图像（Persistence Image）** 向量化。
4. **特征降维与融合**：
   - 将持久性图像展平，拼接并通过 PCA 降维以构建拓扑特征向量 \(v_{TSH}\)。  
   - 将其与传统眼动特征向量 \(v_{BL}\) 融合成联合特征向量 \(v_j,k\)。  
5. **分类模型**：
   - 使用支持向量机（SVM）与随机森林（RF）作为基线分类器。  
   - 构建三类模型：
     - **Baseline**：仅传统特征。
     - **TSH**：仅拓扑特征。
     - **Hybrid**：两者结合。

## 六、主要结论与发现

- **总体表现**：
  - 混合模型（Hybrid）在所有设置下均优于传统基线；
  - 纯拓扑模型（TSH）性能与基线相近甚至略优。
- **最佳结果**：
  - Reader-level（排除L2读者）ROC AUC = **0.99 ± 0.02**（显著高于Baseline的0.97）。
- **关键发现**：
  - 拓扑特征可独立实现近基线性能；
  - 新滤波方式（斜坡、sigmoid、arctan）普遍优于传统水平滤波；
  - 持久同调能捕捉阅读行为中的几何与时间结构，与传统统计指标互补。
