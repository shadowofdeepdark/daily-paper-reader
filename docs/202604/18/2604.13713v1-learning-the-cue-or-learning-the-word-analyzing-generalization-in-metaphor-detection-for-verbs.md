## 摘要
隐喻检测模型在基准测试中表现优异，但尚不清楚这种表现是反映了可迁移的泛化能力还是词汇记忆效应。为了解决这一问题，我们通过分析 RoBERTa（许多当前最先进系统的共享主干网络）在隐喻检测中的泛化特性，聚焦于使用 VU Amsterdam Metaphor Corpus 的英语动词。我们引入了一种受控的词汇保留设置，即严格将选定目标词元的所有实例排除在微调过程之外，并将模型对这些保留词元的预测与对暴露词元（微调中出现过的动词）的预测进行比较。尽管模型在暴露词元上表现最佳，但在保留词元上仍维持了稳健的性能。进一步分析表明，仅依靠句子上下文即可在保留词元上达到与完整模型相当的性能，而静态的动词级嵌入则无法做到。总体而言，这些结果表明，模型的泛化主要由“学习线索”（可迁移的上下文模式）驱动，而“学习词汇”（特定动词的记忆）在存在词汇暴露时则提供了额外的性能提升。

## Abstract
Metaphor detection models achieve strong benchmark performance, yet it remains unclear whether this reflects transferable generalization or lexical memorization. To address this, we analyze generalization in metaphor detection through RoBERTa, the shared backbone of many state-of-the-art systems, focusing on English verbs using the VU Amsterdam Metaphor Corpus. We introduce a controlled lexical hold-out setup where all instances of selected target lemmas are strictly excluded from fine-tuning, and compare predictions on these Held-out lemmas against Exposed lemmas (verbs seen during fine-tuning). While the model performs best on Exposed lemmas, it maintains robust performance on Held-out lemmas. Further analysis reveals that sentence context alone is sufficient to match full-model performance on Held-out lemmas, whereas static verb-level embeddings are not. Together, these results suggest that generalization is primarily driven by "learning the cue" (transferable contextual patterns), while "learning the word" (verb-specific memorization) provides an additive boost when lexical exposure is available.

- **核心问题**：现有模型的高性能究竟源于：
  - **真正的语义泛化（learning the cue）**——捕捉可迁移的上下文线索；
  - 还是 **词汇记忆（learning the word）**——记住特定动词的隐喻倾向性？
- **研究目标**：评估 RoBERTa 在隐喻检测任务中的泛化机制，探究其对未见动词的表现是否依旧稳健，并分析模型依赖的语境与词汇信息来源。

## 二、方法论与技术路线

### 1. 核心思想

- 提出一个 **“词汇排除实验（Lexical Hold-Out Setup）”**：
  - 将选定的动词（lemma）及其所有实例 **完全从训练数据** 中移除；
  - 在测试阶段评估这些未见动词（Held-out lemmas）的隐喻检测性能；
  - 对比它们与训练中出现过的动词（Exposed lemmas）的表现差异；
  - 从而区分“上下文线索泛化”与“词汇记忆”。

### 2. 模型结构

- 使用 **RoBERTa-base** 作为编码器。
- 输入：句子与标注的目标动词。
- 输出：二分类（字面 literal / 隐喻 metaphorical）。
- 模型处理流程：
  1. 文本经过 RoBERTa 编码；
  2. 抽取目标动词的子词隐藏状态；
  3. 通过 **平均池化（mean pooling）** 得到 **Contextualized Target Representation**；
  4. 输入线性分类层 → Softmax 得出概率。
- 关键思想：仅利用句子上下文学习隐喻性的表示，而非硬编码词汇知识。

### 3. 分析维度

- 三种模型变体：
  1. **Full Model**：原始模型（上下文 + 词汇信息）。
  2. **Context-only**：将动词替换为 `<mask>`，仅保留上下文信息，观察“语境信号”的泛化能力。
  3. **Word-only**：利用静态词嵌入训练逻辑回归分类器，测试动词身份单独的预测力。

## 六、主要结论与发现

4. **语义空间结构**：

5. **双重机制**：
