## 摘要
超越传统的多模态讽刺检测二元分类范式，多模态讽刺目标识别（MSTI）提出了更具挑战性的问题，它要求准确定位细粒度的目标，例如文本短语和视觉区域。现有方法主要依赖隐式的跨模态对齐，因而在可解释性和细粒度定位方面表现有限。为克服这些不足，我们提出了 GRASP（Grounded Chain-of-Thought ReAsoning with Dual-Stage Optimization for Multimodal Sarcasm Prediction and Target Identification），一个将视觉具身化与显式链式推理（CoT）相结合的框架，以突破 MSTI 的黑箱限制。具体而言，我们构建了 MSTI-MAX 数据集，该数据集减轻了类别不平衡问题并增强了多模态讽刺线索。我们引入了具身 CoT 推理，它在推理轨迹中显式地锚定与讽刺相关的视觉区域，并促使模型在预测最终分类标签和讽刺目标之前给出推理依据。此外，我们采用了双阶段的结果监督联合优化策略：在坐标感知加权损失下的监督微调，以及细粒度目标策略优化。大量实验表明，GRASP 在跨模态细粒度讽刺目标识别方面超越了现有基线方法，并通过“LLM-as-a-Judge”评估定量衡量了内部推理链的质量。我们的数据集和源代码将发布在 GitHub 上。

## Abstract
Moving beyond the traditional binary classification paradigm of Multimodal Sarcasm Detection, Multimodal Sarcasm Target Identification (MSTI) presents a more formidable challenge, requiring precise localization of fine-grained targets such as textual phrases and visual regions. Existing approaches predominantly rely on implicit cross-modal alignment, offering limited interpretability and suboptimal fine-grained localization. To address these limitations, we propose GRASP, Grounded Chain-of-Thought ReAsoning with Dual-Stage Optimization for Multimodal Sarcasm Prediction and Target Identification, a framework that integrates visual grounding with explicit Chain-of-Thought (CoT) reasoning to move beyond black-box MSTI. Specifically, we curate MSTI-MAX, a refined dataset that mitigates class imbalance and enriches multimodal sarcasm cues. We introduce Grounded CoT reasoning, which explicitly anchors sarcasm-related visual regions within the reasoning trajectory and prompts the model to articulate rationales before predicting the final classification labels and sarcasm targets. Furthermore, we employ a dual-stage outcome-supervised joint optimization strategy: Supervised Fine-Tuning with a coordinate-aware weighted loss, followed by Fine-Grained Target Policy Optimization. Extensive experiments demonstrate that GRASP outperforms existing baselines in fine-grained sarcasm target identification across modalities, and an LLM-as-a-Judge evaluation quantitatively measures the quality of internal reasoning chains. Our dataset and source code will be released on GitHub.

- **研究动机**：  
  传统的多模态讽刺检测（MSD）通常只进行二元分类，即判断图文内容是否带有讽刺性。然而，这种“整体检测”方式缺乏可解释性，无法精确定位具体的讽刺触发点（例如句中短语或图像中的特定区域）。  
  多模态讽刺目标识别（MSTI）任务要求模型识别并**定位讽刺的细粒度目标**，如文字片段或图像框，从而更加贴近人类认知过程。

  1. 现有模型多采用隐式跨模态融合，缺乏可解释的推理路径；

  3. 通用大型多模态语言模型（如 GPT-4o, Qwen-VL）在细粒度讽刺任务上的表现欠佳，推理逻辑模糊。

- **研究目标**：  
  提出一个能显式展示推理逻辑、具备视觉锚定能力的模型框架，改善多模态讽刺任务的**解释性与定位精度**。

## 二、方法论：GRASP 框架

### 1. 核心思想
GRASP（Grounded CoT Reasoning with Dual-Stage Optimization）通过引入**具身化链式推理（Grounded Chain-of-Thought，CoT）**与**双阶段结果监督优化（Dual-Stage Optimization）**，联合实现“**可解释推理 + 精确视觉定位**”。

### 2. 模型设计结构

#### （1）Grounded CoT 推理机制
- 将传统 CoT 扩展到多模态领域，使模型在预测前显式进行视觉与文本的匹配分析；
- 在推理过程中生成 `<Think>` 块以描述分析过程，包括：
  - 视觉描述：提取图像中的关键物体及其归一化坐标；
  - 文本分析：解析语义与情感色彩；
  - 一致性检查：比较视觉与文本内容是否存在语义矛盾；
- 最终结果输出于 `<Answer>` 块，包括讽刺标签、文本目标与视觉坐标。

#### （2）双阶段优化策略

**阶段一：监督微调（SFT）**
- 通过一个增强的 Qwen3.5-Plus 教师模型生成高质量推理样本；
- 在训练时引入“坐标感知加权损失（Coordinate-aware Weighted Loss）”，提升对视觉定位Token的梯度权重；
  - 损失函数：
    \[
    L_{SFT} = -\sum_{t=1}^{N} \omega_t \log P_{\theta}(y_t|X,Y_{<t})
    \]
    其中，当 token 属于坐标时权重 \(\omega_t = \lambda_{coord} > 1\)。

**阶段二：细粒度目标策略优化（FTPO）**
- 类似强化学习的框架：综合多个奖励信号优化推理质量：
  - 格式正确性奖励 \(R_{fmt}\)
  - 分类精度奖励 \(R_{acc}\)
  - 视觉定位奖励（IoU）\(R_{box}\)
  - 文本匹配奖励 \(R_{txt}\)
  - 冗余惩罚 \(P_{over}\)
- 总奖励：
  \[
  R_{total} = \beta_1R_{fmt} + \beta_2R_{acc} + \beta_3R_{box} + \beta_4R_{txt} - \beta_5P_{over}
  \]
- 结合群体采样与策略梯度更新以强化高质量推理路径。
